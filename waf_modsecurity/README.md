# WAF / ModSecurity

configuration et la gestion du module WAF sur ModSecurity ainsi que l'intégration de HashiCorp Vault pour la gestion des secrets.

## Structure
- `config/` : Fichiers de configuration ModSecurity
- `vault/` : Scripts pour l'intégration HashiCorp Vault
- `scripts/` : Scripts d'installation, de gestion et de monitoring
